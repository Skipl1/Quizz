# Mobile-First Refactoring Skill

## Инструкции
1. **Анализ:** Изучи текущий HTML-элемент и его инлайновые стили.
2. **Viewport:** Убедись, что в документе установлен правильный мета-тег viewport.
3. **Tailwind Mapping:** Замени фиксированную ширину (например, `width: 500px`) на адаптивные классы (например, `w-full max-w-lg`).
4. **Tap Targets:** Увеличь все кнопки и поля ввода до минимального размера 44px по высоте для удобства нажатия пальцем.
5. **Grid/Flex:** Используй `flex-col` для мобильных и `md:flex-row` для десктопов.

## Примеры
- "Переделай карточку ответа из index.html так, чтобы она занимала всю ширину экрана на iPhone, но была в две колонки на мониторе."
- "Сделай сетку вариантов ответов адаптивной через Tailwind."
