# Socket-State Sync Skill

## Инструкции
1. **Context/Hook:** Создай React-контекст или кастомный хук `useQuizSocket`.
2. **Event Listeners:** Настрой слушатели событий (`new-question`, `update-leaderboard`) внутри `useEffect` с обязательной очисткой (`socket.off`) при размонтировании.
3. **Reconnection:** Добавь логику обработки обрыва связи (покажи пользователю статус "Переподключение...").
4. **Optimistic UI:** Если возможно, обновляй интерфейс локально до подтверждения сервером для плавности.

## Примеры
- "Напиши хук для получения текущего вопроса, который корректно отписывается от событий при смене экрана."
- "Добавь индикатор качества соединения в верхний угол экрана игрока."
